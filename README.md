# @crumbly/shared

Shared Zod schemas and TypeScript types for Crumbly apps.

## Build

```bash
npm install
npm run build
```

## Usage

Linked locally from `crumbly-api` and `crumbly-web`:

```bash
npm install ../crumbly-shared
```

## Exports

- `registerTenantSchema`, `loginSchema`, `refreshTokenSchema`
- `themeTokensSchema`, `updateTenantSchema`
- `jwtPayloadSchema`
